plex-text

Ejemplos

Input simple

{{plex | json}}
<input type="text" ng-model="plex.text" plex />

Input con Label y Hint

{{plex | json}}
<input type="text" ng-model="plex.text" label="Ingrese un texto" hint="Recuerde no olvidarse el texto" plex="int" />

Opciones

Nombre Tipo Default Descripción
label string Agregar un elemento <label> antes del input
hint string Agrega un texto debajo del input
