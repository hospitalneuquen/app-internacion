plex-select

Ejemplos

Selección de una opción de un array

{{plexSelect | json}}
<input type="hidden" ng-model="plexSelect.seleccion" plex plex-select="item.nombre for item in plexSelect.opciones" />

Selección de múltiples opciones

{{plexSelectMultiple | json}}
<input type="hidden" ng-model="plexSelectMultiple.seleccion" data-multiple="true" plex plex-select="item.nombre for item in plexSelectMultiple.opciones" />

Llamando a una API de países

{{plexSelectAPI | json}}

Opciones

Nombre Tipo Default Descripción
multiple boolean false Permite seleccionar múltiples opciones
timeout int 300 Tiempo de espera antes de buscar las opciones
allowClear boolean false Permite borrar la opción seleccionada con una X
min int 0 Mínimo de caracteres que deben ingresarse antes de buscar las opciones
